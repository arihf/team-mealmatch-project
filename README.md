# team-mealmatch-project
Our awesome CS101 final project

## Our User Stories

### Must Have
- [ ] As a user, I want to enter ingredients I already have, so that I can get recipe options that match them.
- [ ] As a user, I want to see a list of recipes based on my ingredients, so that I can decide what to cook.
- [ ] As a user, I want to view step-by-step instructions, so I can follow the recipe easily.
- [ ] As a user, I want to filter recipes by dietary preferences (vegan, gluten-free, etc.), so I can find meals that fit my needs.

### Nice to Have 
- [ ] As a user, I want to save my favorite recipes, so I can access them quickly later.
- [ ] As a user, I want the app to create a shopping list for missing ingredients, so I can plan meals more efficiently.
- [ ] As a user, I want meal-prep suggestions for the week, so I can plan meals ahead of time.
- [ ] As a user, I want personalized suggestions based on my cooking history, so the app becomes smarter over time.
